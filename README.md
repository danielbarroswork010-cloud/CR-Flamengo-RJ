# Flamengo Match Analytics

Projeto de banco de dados relacional em MySQL para análise de desempenho do Clube de Regatas do Flamengo.

## Tecnologias
- MySQL
- SQL Avançado
- Modelagem Relacional

## Funcionalidades
- Registro de jogos
- Classificação automática de resultados (trigger)
- Views analíticas
- Stored Procedures
- Índices para performance
- Estatísticas de vitórias, empates e derrotas

## Conceitos aplicados
- Normalização
- Regras de negócio no banco
- Performance com índices
- Análise de dados esportivos

## Possíveis evoluções
- API REST
- Dashboard BI
- Integração com frontend
